# プロジェクトコンセプト

## プロジェクト名

**RigLab** (仮)

## 概要

PC初心者から自作経験者まで、誰でも簡単にPC構成を作成・比較できるWebアプリケーション。

## ターゲットユーザー

| ユーザー層 | ニーズ |
|-----------|--------|
| PC初心者 | 何を選べばいいかわからない → おまかせ構成で解決 |
| PC自作したい人 | 自分でパーツを選びたい → カスタム構成で実現 |

## コア機能

### 1. おまかせ構成提案

予算・用途から最適なPC構成を提案する。

**入力**:
- 予算（例: 10万円、15万円、20万円）
- 用途（例: ゲーム、動画編集、事務作業、プログラミング）
- または両方の組み合わせ

**出力**:
- 最適なパーツ構成
- 合計価格
- 性能の目安

### 2. カスタム構成作成

ユーザーが自由にパーツを選んでPC構成を作成する。

**機能**:
- パーツ一覧から選択（CPU、GPU、メモリ、ストレージ等）
- 合計価格の自動計算
- 構成の共有（URLエンコード + OG画像生成）
- 構成の保存（ログイン時のみ）

## 認証

- **ログインなし**: 構成作成・閲覧・共有が可能
- **ログインあり**: 構成の保存・履歴管理が可能

## 技術スタック

| 層 | 技術 |
|----|------|
| Frontend | Next.js 15 (App Router) + Tailwind CSS |
| Backend | Rails 7.1 API |
| Database | MySQL 8.0 |
| Auth | NextAuth.js (Auth.js) |
| OG画像生成 | next/og (ImageResponse) |

> **セキュリティ注意**: Next.js 15.2.3以上を使用すること（CVE-2025-29927対策）

## パーツデータ

### Phase 1（初期リリース）
- **手動登録**: 管理画面からパーツ情報を登録
- 主要パーツを事前登録しておく

### Phase 2（機能追加）
- **価格.com API連携**: パーツ情報・価格を自動取得
  - 参考: https://gist.github.com/fuji44/2826ef43e0ce1ebf65ccf5db483dc411
- **Keepa API連携**: Amazon価格推移の取得（要課金）
- 最新価格の自動更新

## 成功指標

- ユーザーが3ステップ以内で構成を作成できる
- 初心者でも迷わず使える UI
